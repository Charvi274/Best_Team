# Hackathon Squad

Heuristic-based C++ solution for the Hackathon Squad optimization problem.

## Approach
- Greedy selection using:
  
  skill / (degree + 1)

- Local search optimization
- Randomized improvements

## Tech Used
- C++
- Graph Algorithms
- Greedy Heuristics

## Files
- `hs_code.cpp` → main solution code
- `input_hs.txt` → sample input dataset
